# Changelog

## 0.0.3

- Updated the extension icon to a more fitting design that reflects the leveling up theme.
- Fixed save XP so pressing Ctrl+S on already-saved files no longer grants rewards.
- Improved dashboard refresh reliability when the Level Up view becomes visible again.
- Fixed duplicate dashboard event bindings that could make the webview behave inconsistently over time.

## 0.0.2

- Fixed dashboard progress meters so they render correctly under the webview content security policy.
- Rebalanced dashboard trait progress bars so they reflect different coding patterns more accurately.
- Improved progress bar precision for level and achievement tracking.

## 0.0.1

- Initial MVP for the Level Up extension.
- XP gains from edits and saves.
- Levels, streaks, achievement unlocks, and persistent progress.
- Status bar summary and Activity Bar dashboard.
